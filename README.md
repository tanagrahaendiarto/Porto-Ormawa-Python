# Kalkulator Sederhana
Program ini merupakan kalkulator sederhana berbasis terminal yang dibuat sebagai bagian dari tugas portofolio pendaftaran Laksana Ormawa Eksekutif Biro Riset dan Pengembangan 2025/2026.

# Deskripsi Program
Kalkulator ini memungkinkan pengguna melakukan perhitungan aritmatika dasar melalui menu yang ditampilkan di terminal. Program dibuat untuk melatih logika pemrograman, cara menerima input dari pengguna, serta proses pengolahan data secara sederhana dan terstruktur.

# Bahasa Pemrograman
Python 3

# Fitur
1. Operasi penjumlahan
2. Operasi pengurangan
3. Operasi perkalian
4. Operasi pembagian
5. Operasi modulus
6. Menu pilihan operasi
7. Opsi untuk keluar dari program

# Alur Program
1. Program menampilkan daftar menu operasi.
2. Pengguna memilih operasi yang diinginkan.
3. Pengguna memasukkan dua angka.
4. Program melakukan perhitungan sesuai pilihan.
5. Hasil perhitungan ditampilkan.
6. Program kembali ke menu atau berhenti sesuai pilihan pengguna.

# Cara Menjalankan Program

Pastikan Python sudah terinstal di perangkat, kemudian jalankan perintah berikut melalui terminal atau command prompt:

python kalkulator.py
